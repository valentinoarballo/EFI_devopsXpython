API con Flask y Docker

Este proyecto es una API de Flask que usa Docker y MySQL para crear y gestionar usuarios, publicaciones, temas y comentarios. La API permite hacer operaciones CRUD (crear, leer, actualizar y borrar) sobre estos recursos, usando diferentes endpoints y métodos HTTP. La API también usa SQLAlchemy, Marshmallow y pytest para facilitar el trabajo con la base de datos, la serialización y deserialización de los datos, y los tests de la aplicación.



Requisitos

Para ejecutar este proyecto, necesitas tener instalado Docker y docker-compose en tu máquina. También necesitas tener un archivo .env con las variables de entorno necesarias para la configuración de la aplicación y la base de datos. Un ejemplo de este archivo es el siguiente:

SQLALCHEMY_DATABASE_URI = mysql+pymysql://db_user:root@mysql_db:3306/db_valentino
 MYSQL_PASSWORD = root
 MYSQL_USER = db_user
 MYSQL_DATABASE = db_valentino
 FLASK_APP=app

Los datos que se reciben en las peticiones son de tipo JSON, y tienen el siguiente formato:
