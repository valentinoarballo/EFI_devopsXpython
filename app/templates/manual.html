Instrucciones para el uso de la API en Postman

Para tener todas las requests con todos sus metodos tenemos que hacer lo siguente



Paso 1

Dentro de Postman tenemos que importar las colecciones que se encuentran en formato JSON dentro de el directorio raiz de este proyecto.
Para esto, esta el boton Import arriba a la izquierda.



Paso 2

Esto nos va a abrir una ventana que nos va a solicitar que marquemos la carpeta o archivos con las colecciones JSON.
Aca podes elegir lo que quieras yo pongo la opcion Files pero da igual.



Paso 3

Tenemos que seleccionar los 4 archivos si elegimos Files o la carpeta entera si elegimos Folders



Paso 4

El paso final es importarlo



Paso 5

Ya tendrian que aparecer todas las peticiones separadas por carpetas con cada uno de los metodos disponibles para la API
